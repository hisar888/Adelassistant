# Adel Assistant — Семейный ИИ-наставник

Микро-PWA на React + Tailwind CSS v4, разработанное для обучения детей и взрослых навыкам работы с ИИ через квесты, чаты и геймификацию.

## 🔥 Как запустить приложение

Для просмотра приложения в браузере выполните следующие команды в терминале:

1. **Перейдите в папку проекта:**
```bash
cd "Desktop\Adel Assitant"
```

2. **Запустите сервер разработки:**
```bash
npm run dev
```

3. **Откройте ссылку в браузере (Ctrl+Click):**
Обычно это `http://localhost:5173/`

## ✨ Основные функции

- **Интерактивная карта навыков**: Продвигайся по уровням — от основ ИИ до сложной автоматизации.
- **ИИ-наставник**: Умный помощник (OpenRouter API), способный говорить простым языком.
- **Движок квестов**: Обучение через диалоги, викторины и практическую работу с промптами.
- **Семейный лидерборд**: Соревнуйся с близкими, набирай XP и открывай достижения.
- **Инвентарь**: Сохраняй полезные шаблоны и подсказки.
- **Поддержка PWA**: Установи приложение на рабочий стол смартфона.
- **Темная тема**: Комфортное обучение в любое время суток.

## 🛠 Технологический стек

- **Frontend**: React 19, Vite 6, Tailwind CSS v4.
- **Animations**: Framer Motion.
- **State**: Zustand.
- **Backend Ready**: Firebase Functions + Hosting.
- **Localization**: i18next (RU, TT, EN).
